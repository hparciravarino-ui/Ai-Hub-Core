# Environment Variables

* **GEMINI_API_KEY**
  * Obbligatoria: Sì (per GeminiService server-side embedding).
  * Default: Assente.
  * Modulo: `src/core/services/GeminiService.ts`

* **PORT**
  * Obbligatoria: No.
  * Default: 3000.
  * Modulo: `server.ts`

* **NODE_ENV**
  * Obbligatoria: No.
  * Default: `development` o `production`.
  * Modulo: `server.ts`
